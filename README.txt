# Semantic Portfolio Template – by Roger

A simple, responsive portfolio built with **semantic HTML + CSS only** (no `<div>` elements).  
Includes **Light/Dark mode** toggle and clean, accessible structure.

## Files
- `index.html` (one-page template)
- `extras/color-themes.css` (optional color palettes)

## Quick Start (5 minutes)
1) Open `index.html` in VS Code.
2) Replace text: your name, email, and project descriptions.
3) Save and double-click `index.html` to view in your browser.

## Colors & Branding
Colors are defined with CSS variables (`:root`).  
To switch palettes, copy a block from `extras/color-themes.css` into your `<style>` or include it as an extra `<link>` and override variables.

## Free Hosting (no code)
- **Netlify**: Drag your folder to https://app.netlify.com/drop → live.
- **GitHub Pages**: New repo → upload files → Settings → Pages → deploy from main.

## Contact Form
This is a static template. To receive form submissions, connect a service like **Formspree** or **Netlify Forms**.
